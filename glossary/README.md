# glossary

Terminology and concept definitions for uips-browser-kit.

## What belongs here

- Definitions of domain terms (browser automation, UiPath, TOGAF layers)
- Disambiguation of terms used differently across contexts
- Cross-references to ADRs or guides where a term is first defined

## File naming

One file per letter or topic group, e.g. `a-c.md`, `browser-terms.md`.
Prefer alphabetical grouping for long glossaries.
