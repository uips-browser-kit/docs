# togaf/tec — Technology Architecture

Inventories for the Technology Architecture layer.

## What belongs here

- Physical Technology Component (PTC) inventory — `physical-technology-components.csv`
- Logical Technology Component inventory
- Technology standards and platform choices

## Column schema (PTC)

`id, category, name, description, vendor, version, source, productname, platform,
licence, created_at, reviewed_at, retired_at, path, visibility, nickname`
